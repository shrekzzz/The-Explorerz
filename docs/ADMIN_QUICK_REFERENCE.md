# Admin Quick Reference - Enquiry Management

## 🚀 Quick Start

1. **Login**: Navigate to `/login` with your admin credentials
2. **Dashboard**: Click "Enquiries" tab in the sidebar
3. **View**: See all enquiries in the table
4. **Filter**: Use status dropdown to filter by NEW, CONTACTED, CONVERTED, or CLOSED
5. **Update**: Click status badge to change enquiry status

## 📊 Status Workflow

```
🔵 NEW → 🟡 CONTACTED → 🟢 CONVERTED → ⚫ CLOSED
```

### When to Use Each Status

| Status | When to Use | Color |
|--------|-------------|-------|
| **NEW** | Fresh enquiry, not yet contacted | Blue |
| **CONTACTED** | You've reached out to the customer | Amber |
| **CONVERTED** | Customer has booked the package | Green |
| **CLOSED** | Enquiry resolved (booked or declined) | Gray |

## 🎯 Daily Workflow

### Morning Routine
1. Check "NEW" enquiries
2. Contact customers via email/phone
3. Update status to "CONTACTED"
4. Add notes (if feature available)

### Throughout the Day
1. Monitor "CONTACTED" enquiries
2. Follow up with customers
3. Update to "CONVERTED" when booked
4. Move to "CLOSED" when resolved

### End of Day
1. Review pending "NEW" enquiries
2. Check response times
3. Plan follow-ups for next day

## 🔍 Filtering & Search

### Filter by Status
1. Click status dropdown at top
2. Select: All, NEW, CONTACTED, CONVERTED, or CLOSED
3. Table updates automatically

### Quick Filters (Keyboard Shortcuts)
- `Ctrl+1`: Show all enquiries
- `Ctrl+2`: Show NEW only
- `Ctrl+3`: Show CONTACTED only
- `Ctrl+4`: Show CONVERTED only
- `Ctrl+5`: Show CLOSED only

*(Note: Keyboard shortcuts may need to be implemented)*
